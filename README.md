# NXMSG — Secure Messenger

Минималистичный зашифрованный мессенджер с уникальными ID и QR-кодами.

## Стек
- **Backend**: Node.js + Express + WebSocket (ws)
- **Frontend**: Vanilla HTML/CSS/JS (без фреймворков)
- **Шифрование**: AES-256-GCM (сервер), санитизация входных данных

## Запуск

```bash
cd messenger
npm install
npm start
# → http://localhost:3000
```

## Безопасность

### ✅ Защита от XSS (уязвимость типа «Max Messenger»)
- **Весь пользовательский контент рендерится через `textContent`**, а не `innerHTML`
- Это полностью исключает выполнение JS-кода из сообщений
- Дополнительная HTML-санитизация на сервере (`&lt;`, `&gt;`, `&quot;` и т.д.)
- Content-Security-Policy заголовки на сервере

### ✅ Шифрование сообщений
- AES-256-GCM с уникальным IV для каждого сообщения
- Ключ выводится из пары ID через HMAC-SHA256
- Auth tag для проверки целостности

### ✅ Уникальные ID (~20 символов)
- Криптографически случайные (crypto.randomBytes)
- Алфавит: A-Z, a-z, 0-9, !@#$%^&*-_+=~
- Валидация на клиенте и сервере

### ✅ Безопасные HTTP-заголовки
- X-Content-Type-Options: nosniff
- X-Frame-Options: DENY
- X-XSS-Protection: 1; mode=block
- Content-Security-Policy

### ✅ Список чатов
- Показываются ТОЛЬКО начатые чаты (нет глобального списка пользователей)
